# :computer: How to run

```bash
# Clone Repository
$ git clone https://github.com/gustavogk/ignite-timer.git
```

```bash
# Install Dependencies
$ npm install

# Run Aplication
$ npm run dev
```
Go to http://localhost:3000/ to see the result.
